---
title: OpenAPI Specification
---

The static OpenAPI specification is generated into the repository root:

- [`openapi.yaml`](https://github.com/stritti/nocodb-middleware/blob/main/openapi.yaml)
- [`openapi.json`](https://github.com/stritti/nocodb-middleware/blob/main/openapi.json)

Regenerate it with:

```bash
npm run build && npm run generate:openapi
```
